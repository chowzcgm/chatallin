# Roadmap

This roadmap describes planned milestones for chatallin.

## v0.2.0 - Core Runtime MVP
- Backend scaffold (TypeScript + Fastify)
- Provider abstraction and registry
- OpenAI/Gemini/DeepSeek basic adapters
- Moderator multi-round orchestration
- Structured result output (conclusion/disagreements/evidence/next actions)

## v0.3.0 - Web Collaboration MVP
- Web session creation page
- User-defined model selection
- Transcript and replay view
- Basic cost/usage display
- Configurable rounds and stop conditions

## v0.4.0 - Open Source Usability
- Config import/export (JSON templates)
- Better docs and onboarding
- Example presets for common workflows
- Improved testing and CI

## v0.5.0 - Reliability and Ecosystem
- Better retries, timeouts, fallback strategies
- Plugin-style provider extension guide
- Community contribution examples
- Release automation improvements
